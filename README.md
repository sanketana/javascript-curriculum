# javascript-curriculum

The JavaScript track for **Sanketana School of Code** — two 24-session modules taught 1:1 for middle and high school students.

- **[Beginner](beginner/)** — no prior JavaScript; add interactivity to static pages and build simple browser games and utilities. *(Coming soon.)*
- **[Advanced](advanced/00-course-overview.md)** — a foundation for full-stack web development and frameworks (React, Node, Express, Next). *(In progress.)*

Everything runs in the browser — plain HTML, CSS, and JavaScript, no build step. See each module's `00-course-overview.md` for the full syllabus.
